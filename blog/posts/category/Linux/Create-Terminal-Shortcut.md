# CentOS 8 터미널 단축키 만들기

CentOS 에서는 우분투와 다르게 터미널 단축키가 없다고 한다.  
매번 마우스로 메뉴를 클릭해서 터미널을 여는것은 불편하므로 단축키를 등록해보겠다.  

### 1. 키보드 설정 열기.
### 2. 단축키 정보 입력 후 등록하기.
```
Name : Terminal shortcut (임의 설정)
command : gnome-terminal
set short cut : Ctrl + Alt + T (임의설정)
```
